GTR의 히어로 영상(MANO + 로스팅 + 공간)은 % Arabica 접근과 동일 철학.
차이: GTR은 3가지 차별점을 한 영상에 압축 — 편집 퀄리티가 전부.

Track A — 핵심 인사이트

5가지 핵심 배움

1. B&W 클린 + 따뜻함의 균형
이미지 톤에서 따뜻함, 레이아웃/타이포는 차갑게 유지
2. 히어로 영상 = 최강 무기
씬 전환 호흡, 색보정 일관성이 전체 인상을 결정
3. 풍미 노트 그래픽 = 미개척 영역
모든 경쟁사가 텍스트 나열. 인터랙티브 시각화로 업계 최초 차별화
4. 카페 메뉴 → 상품 순서는 합리적
매장 방문 유도 1차 목표에 부합하는 현재 섹션 순서
5. 인터랙션 수준은 이미 경쟁력 있음
blur→sharp, fade-in stagger 등 Onyx에 근접. 과하지 않게 통제만 잘 하면 목표 달성 가능.

Track D — 핵심 인사이트

세그먼트별 니즈 매트릭스

카페 메뉴가 가장 많은 세그먼트에 영향
단골 + 디저트팬 + 입문자 = 전체의 ~70%
카페 메뉴 퀄리티가 웹사이트 전체 만족도를 좌우
버터떡 = 현재 가장 강력한 매장 방문 트리거
풍미 시각화는 ~20%지만 차별화 극대
홈브루어 세그먼트가 재구매 + 구독 전환의 핵심
"이 브랜드 진짜 신경 쓴다"는 인상을 모든 세그먼트에 전달
브랜드 스토리는 첫 방문자에게 집중
선물 구매자 + 입문자에게 신뢰 형성의 핵심
메인에서 가볍게 노출 + 서브 페이지에서 깊이
SNS ↔ 웹사이트 = 진입의 핵심 축
인스타그램이 가장 활발한 채널
인스타 → 웹사이트 → 매장의 흐름이 주요 여정

Design Direction — 인터랙션

인터랙션 로드맵

단계적·점진적 적용. 기본 효과 먼저, 느낌을 보면서 고급 효과 추가.

Phase 1
스크롤 리빌 (fade-in + translateY, 0.6s) · Stagger (100ms 간격 순차 등장) · Blur→Sharp (1회성 이미지 리빌)
Phase 2
스크롤 연동 동적 블러 — Onyx 참고. 뷰포트 진입/이탈 시 블러↔선명 반복. 1회성이 아닌 스크롤 위치에 따라 계속 반응
Phase 3
고급 스크롤 기반 효과 — Phase 2 적용 후 체감 확인하고 결정

Design Direction — UX 원칙

7가지 UX 설계 원칙

01
보여주되, 설명하지 않는다
이미지·영상이 텍스트보다 먼저. 갓 구운 빵 클로즈업 한 장 > "매일 빵을 굽습니다"
02
여유로운 페이싱
"take time" 체현. 120px+ 여백, 0.6s+ 리빌, 팝업 남발 금지
03
감각의 트리거
냄새·촉감을 시각으로 환기. 소재감 클로즈업, MANO 추출, 공간
04
정보의 계층화
첫 눈에 핵심만, 관심 있는 사람만 더 깊이. 카드 → 상세 → 추가
05
구매 허들 최소화
분쇄 옵션 4종, 구독 토글, 사이드 드로어 장바구니
06
매장 방문 = 궁극의 CTA
모든 경로가 "가보고 싶다"로 수렴. 메뉴(상단) → Visit(하단)
07
디테일이 신뢰를 만든다
호버 섬세함, 자간 조정, 풍미 시각화 — 작은 정성 = "여기 진짜 신경 쓴다" = 웹사이트 퀄리티 = 브랜드 퀄리티
