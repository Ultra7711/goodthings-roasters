# Good Things Roasters — 리서치 계획서 v1

> **목적:** 국내외 스페셜티 커피 브랜드와 견줄 수 있는 수준의 웹사이트 구축을 위한 종합 리서치
> **작성일:** 2026-04-05
> **리서치 유형:** 종합 (경쟁사 벤치마킹 + 타겟 유저 + UX 벤치마킹 + 브랜드 포지셔닝)

---

## 1. 프로젝트 배경

### 현재 상황
- 카페24 기반 웹사이트 운영 중 — 플랫폼 한계로 인한 디자인·기능 제약
- 메인 히어로 섹션 오류 상태 등 기본적인 완성도 미달
- 브랜드 퀄리티 대비 웹사이트가 브랜드 가치를 반영하지 못하는 상태

### 목표
- 카페24 탈피 → 자체 구축 웹사이트 (Next.js + Supabase + Vercel)
- 국내외 스페셜티 커피 브랜드 웹사이트 최상위 수준의 퀄리티 달성
- 온라인에서도 GTR의 브랜드 경험을 전달할 수 있는 웹사이트

### GTR 핵심 차별점
| 차별점 | 설명 |
|--------|------|
| **좋은 원두** | 스페셜티 등급 원두 사용, 품질에 대한 진정성 |
| **마노스(MANO) 머신** | 프리미엄 추출 머신 — 기술과 맛의 결합, 장비에 대한 자부심 |
| **자체 베이커리** | 직접 매장에서 굽는 빵·디저트 — 갓 구운 빵 냄새가 매장에 머무는 감각적 경험 |
| **공간 경험** | 커피 + 베이킹 향 + 공간이 만드는 총체적 감각 경험 |

## 2. 리서치 구조

총 4개 트랙으로 구성하며, 각 트랙은 병렬 진행 가능하다.

```
Track A: 경쟁사 벤치마킹 ──┐
Track B: UX 벤치마킹 ──────┤──→ 종합 인사이트 ──→ 디자인 방향성 확정
Track C: 브랜드 포지셔닝 ──┤
Track D: 타겟 유저 리서치 ─┘
```

---

## Track A. 경쟁사 벤치마킹

### 목적
국내외 스페셜티 커피 브랜드 웹사이트의 디자인·기능·콘텐츠 수준을 파악하고, GTR이 도달해야 할 기준선(baseline)을 설정한다.

### 분석 대상

#### 국내 (6~8개)
| 브랜드 | 선정 이유 |
|--------|----------|
| 프릳츠 커피 | 국내 스페셜티 대표, 브랜드 아이덴티티 강함 |
| 블루보틀 코리아 | 글로벌 브랜드의 한국 로컬라이징 |
| 펠트 커피 | 미니멀 브랜딩, 온라인 커머스 강점 |
| 커피 리브레 | 다이렉트 트레이드, 원두 스토리텔링 |
| 나무사이로 | 로스터리 + 카페 + 온라인 판매 모델 |
| 모모스 커피 | 부산 기반, 지역 스페셜티 대표 |
| 앤트러사이트 | 공간 경험 + 브랜드 디자인 |
| 매뉴팩트 커피 | 로스팅 전문, 구독 모델 운영 |

#### 해외 (5~7개)
| 브랜드 | 선정 이유 |
|--------|----------|
| Blue Bottle Coffee | 글로벌 벤치마크, 미니멀 UX |
| Stumptown Coffee | 원두 스토리텔링, 구독 모델 |
| Counter Culture Coffee | 교육 콘텐츠, 투명성 |
| Onyx Coffee Lab | 웹사이트 퀄리티 최상위, 비주얼 강점 |
| Verve Coffee Roasters | 라이프스타일 브랜딩, 이커머스 UX |
| % Arabica | 미니멀 극단, 공간 브랜딩 |
| Square Mile Coffee | 유럽 스페셜티, 정보 중심 설계 |

| 카테고리 | 세부 항목 |
|---------|----------|
| **비주얼 디자인** | 색상 팔레트, 타이포그래피, 이미지 스타일, 여백 활용, 전체 톤앤무드 |
| **레이아웃 구조** | 메인 페이지 섹션 구성, 네비게이션 구조, 페이지 뎁스(depth) |
| **히어로 섹션** | 유형(비디오/이미지/슬라이더), 메시지, CTA 배치 |
| **상품 페이지** | 상품 카드 디자인, 필터/정렬, 상세 페이지 구성, 원두 정보 표시 방식 |
| **브랜드 스토리** | 스토리텔링 방식, 콘텐츠 뎁스, 철학/가치 전달 방법 |
| **카페/매장 정보** | 매장 소개 방식, 지도, 메뉴 표시, 공간 사진 활용 |
| **구독/정기배송** | 구독 모델 유무, 플랜 구성, 가격 표시, 가입 플로우 |
| **모바일 경험** | 반응형 퀄리티, 모바일 전용 UX, 터치 인터랙션 |

### 산출물

- 벤치마크 인사이트 리포트 (마크다운)
- GTR 기준선(baseline) 정의 문서

## Track B. UX 벤치마킹

### 목적
커피 이커머스의 핵심 사용자 플로우를 분석하고, GTR 웹사이트의 UX 설계 기준을 수립한다.

### 분석 대상 플로우

#### 1. 상품 탐색 → 구매 플로우
```
메인 진입 → 상품 목록 → 필터/정렬 → 상품 상세 → 옵션 선택 → 장바구니 → 결제
```
- 원두 상품의 정보 표시 (로스팅 레벨, 원산지, 풍미 노트, 무게/분쇄 옵션)
- 비커피 상품(베이커리/디저트)의 표시 방식
- 장바구니 인터랙션 (사이드 드로어 vs 별도 페이지)
- 게스트 결제 vs 회원 결제

#### 2. 구독/정기배송 플로우
```
구독 소개 → 플랜 선택 → 원두 선택 → 배송 주기 → 결제 → 관리
```
- 구독 가치 제안(value proposition) 전달 방식
- 플랜 비교 UI
- 구독 관리 (일시정지/변경/해지)

#### 3. 카페 메뉴 탐색 플로우
```
메뉴 진입 → 카테고리 선택 → 메뉴 상세 → 매장 정보/위치
```
- 온라인 메뉴판의 정보 구조
- 메뉴 사진 활용 방식
- 매장 주문과의 연계

#### 4. 브랜드 스토리 탐색 플로우
```
메인 CTA → 스토리 페이지 → 철학/원두/로스팅/공간 섹션
```
- 스크롤 스토리텔링 vs 페이지 분리
- 이미지/비디오 콘텐츠 비중
- 감성적 전달 vs 정보적 전달 밸런스

### 분석 항목
| 항목 | 설명 |
|------|------|
| 태스크 완료 스텝 수 | 주요 태스크별 클릭/스텝 수 비교 |
| 정보 위계 | 각 단계에서 사용자에게 제공되는 정보의 우선순위 |
| 이탈 리스크 포인트 | 사용자가 이탈할 가능성이 높은 지점 |
| 차별화 패턴 | 경쟁사에서 발견되는 독특하고 효과적인 UX 패턴 |
| 공통 패턴 | 대부분의 브랜드가 채택하는 표준 패턴 (= 사용자 기대치) |

### 산출물
- 플로우별 비교 분석표
- UX 패턴 라이브러리 (스크린샷 + 설명)
- GTR UX 설계 원칙 초안

## Track C. 브랜드 포지셔닝

### 목적
GTR의 브랜드 포지션을 경쟁사 대비 명확히 정의하고, 웹사이트가 전달해야 할 핵심 브랜드 메시지를 도출한다.

### 분석 프레임워크

#### 1. 포지셔닝 맵
두 개의 축으로 경쟁사를 배치하고 GTR의 위치를 정의한다.

```
                    프리미엄/고급
                         │
                         │
    전통/클래식 ──────────┼────────── 모던/트렌디
                         │
                         │
                    대중적/접근성
```

#### 2. 브랜드 속성 분석
| 속성 | GTR 현재 | GTR 목표 | 비고 |
|------|---------|---------|------|
| 톤앤무드 | 카페24 한계로 미전달 | 모던·정갈한 B&W 프리미엄 | 블랙 & 화이트 톤 확정 |
| 핵심 가치 | 좋은 원두 + 좋은 추출 + 좋은 베이커리 | "good things, take time" | 시간을 들여 좋은 것을 만든다는 철학 |
| 감각 경험 | 매장 방문 시 체감 (빵 냄새, 공간) | 온라인에서도 감각적 경험 전달 | **히어로 영상이 핵심 장치** |
| 타겟 인상 | ? | "여기 진짜 신경 쓴다" | 퀄리티에 대한 신뢰감 |

#### 3. GTR 온라인 브랜드 경험 과제
오프라인에서 자연스럽게 전달되는 경험을 온라인으로 변환해야 한다:

> **핵심 장치: 히어로 영상**
> 현재 히어로 영상은 MANO 머신 추출(미래적 차별감) + 원두 로스팅 과정(진정성) + 매장 공간(방문 욕구)을
> 하나의 영상에 압축하여 첫 인상에서 GTR의 모든 차별점을 전달하는 가장 중요한 장치.
> 히어로 영상의 품질과 편집이 웹사이트 전체 인상을 좌우한다.

| 오프라인 경험 | 온라인 변환 방안 | 현재 상태 |
|-------------|---------------|----------|
| MANO 머신 추출 과정 | **히어로 영상** (1차) + 스토리 페이지 (2차) | ✅ 히어로 영상에 포함 |
| 원두 로스팅 과정 | **히어로 영상** (1차) + 상품 상세 (2차) | ✅ 히어로 영상에 포함 |
| 매장 공간의 분위기 | **히어로 영상** (1차) + 갤러리 (2차) | ✅ 히어로 영상에 포함 |
| 갓 구운 빵 냄새 | 베이킹 과정 사진, "오늘의 베이킹" 콘텐츠, SNS 연동 | 🔲 추가 콘텐츠 필요 |
| 바리스타와의 대화 | 원두 추천 가이드, 브루잉 팁 콘텐츠 | 🔲 추가 콘텐츠 필요 |
| 커피 한 잔의 여유 | 웹사이트 자체의 여유로운 페이싱 (급하지 않은 UX) | ✅ 프로토타입에 이미 반영 중 |

### 산출물
- 경쟁사 포지셔닝 맵
- GTR 브랜드 속성 정의서
- 웹사이트 브랜드 메시지 프레임워크
- 오프라인→온라인 경험 변환 전략 문서

## Track D. 타겟 유저 리서치

### 목적
GTR의 핵심 고객층을 정의하고, 그들의 온라인 행동 패턴과 니즈를 파악한다.

### 리서치 방법

#### 1. 기존 고객 데이터 분석 (가능한 경우)
- 카페24 주문 데이터: 재구매율, 인기 상품, 주문 패턴
- 매장 POS 데이터: 시간대별 방문, 인기 메뉴, 객단가
- SNS 팔로워 분석: 연령대, 관심사, 인게이지먼트

#### 2. 고객 세그먼트 가설
| 세그먼트 | 특성 | 핵심 니즈 | 웹사이트 이용 시나리오 |
|---------|------|----------|---------------------|
| **홈브루어** | 집에서 커피를 내려 마시는 사람 | 원두 정보, 브루잉 가이드, 정기배송 | 원두 탐색 → 구매 → 재구매/구독 |
| **카페 단골** | GTR 매장을 주기적으로 방문 | 메뉴 확인, 신메뉴, 매장 정보 | 메뉴 확인 → 방문 → 굿즈/원두 온라인 구매 |
| **선물 구매자** | 커피를 선물로 보내려는 사람 | 선물 포장, 추천, 가격대별 필터 | 선물 세트 탐색 → 구매 → 배송 |
| **커피 입문자** | 스페셜티 커피에 관심을 갖기 시작 | 쉬운 설명, 추천, 입문 세트 | 브랜드 탐색 → 추천 원두 → 첫 구매 |
| **디저트 팬** | 베이커리/디저트가 주 목적 | 오늘의 빵, 메뉴, 매장 위치 | 디저트 메뉴 확인 → 매장 방문 |

#### 3. 간이 고객 인터뷰 (선택)
매장 방문 고객 5~8명 대상 비구조화 인터뷰:
- "GTR을 어떻게 알게 되셨나요?"
- "GTR 하면 떠오르는 이미지/느낌은?"
- "온라인으로 원두를 구매할 때 가장 중요하게 보는 것은?"
- "GTR 웹사이트에서 기대하는 것은?"

### 산출물
- 고객 세그먼트 정의서
- 세그먼트별 페르소나 카드
- 사용자 여정 지도 (User Journey Map)
- 인터뷰 인사이트 (실시 시)

## 4. 진행 일정

| 단계 | 트랙 | 작업 내용 | 비고 |

## 부록: 브랜드 메시지 침투 전략 (초안)

> "good things, take time" — 이 메시지가 특정 섹션에만 쓰이는 슬로건이 아니라, 웹사이트 전체의 결에 스며드는 방법.

### 원칙: 말하지 않고 느끼게 한다

슬로건을 반복하는 것이 아니라, **사용자가 웹사이트를 경험하면서 자연스럽게 "여기 뭔가 시간을 들였다"고 느끼게** 만드는 것이 핵심.

### 침투 레이어

| 레이어 | 적용 방식 | 예시 |
|--------|----------|------|
| **① 페이싱 (속도감)** | 급하지 않은 UX — 여유 있는 스크롤 간격, 넉넉한 여백, 섹션 간 호흡 | 섹션 간 120px+ 여백, 콘텐츠 밀도를 의도적으로 낮춤 |
| **② 리빌 타이밍** | 한꺼번에 보여주지 않고 스크롤에 따라 천천히 드러남 | 부드러운 fade-in (0.8s+), 순차 등장 (stagger), blur→sharp |
| **③ 트랜지션 이징** | 모든 애니메이션에 급한 느낌 없는 이징 적용 | `ease-out` 기반, 빠른 snap 동작 지양, 부드러운 감속 |
| **④ 이미지 톤** | 차분한 색온도의 사진 — 과도한 채도나 HDR 지양 | 자연광 기반, 약간 낮은 채도, 따뜻한 톤 보정 |
| **⑤ 카피 톤** | 짧고 여유로운 문장 — 마침표 뒤의 공간도 디자인 | "천천히 내린 한 잔." / "좋은 것엔 시간이 필요합니다." |
| **⑥ 디테일 품질** | 작은 요소까지 정성이 느껴지는 마감 | 호버 효과의 섬세함, 아이콘 디테일, 폰트 자간 미세 조정 |
| **⑦ 콘텐츠 깊이** | 원두·로스팅·추출 과정을 성의 있게 보여줌 | 원두 상세의 풍미 노트 그래픽, 로스팅 프로파일, MANO 추출 과정 |

### 안티패턴 (피해야 할 것)

| 피해야 할 것 | 이유 |
|-------------|------|
| 팝업 남발 | "시간을 들인다"의 반대 — 급하게 전환을 유도하는 느낌 |
| 과도한 모션/패럴랙스 | 정신없으면 "단정함"이 깨짐 |
| 빽빽한 정보 밀도 | 여유와 호흡이 없어짐 |
| 형광색·네온·강한 그라디언트 | 모던 B&W 톤과 충돌 |
| "지금 바로!", "한정 수량!" 식 어조 | 급박함은 브랜드 철학과 정반대 |

## 부록: 감각 경험 온라인 변환 전략 (초안)

> 빵 냄새, 커피 향, 공간의 질감 — 이것들은 스크린을 통해 전달할 수 없다.
> 그래서 **직접 전달이 아니라, 상상하게 만드는 것**이 전략의 핵심.

### 접근: 감각의 트리거

냄새나 촉감을 직접 전달할 수는 없지만, **시각과 텍스트로 그 감각의 기억을 자극**할 수 있다.

| 오프라인 감각 | 변환 전략 | 구체적 실행 |
|-------------|----------|-----------|
| **갓 구운 빵 냄새** | "방금"이라는 시간성 강조 | 베이킹 과정 클로즈업 사진 (반죽→오븐→갓 구운 빵), "매일 아침 굽습니다" 같은 현재진행 카피, SNS 연동으로 오늘의 베이킹 실시간 공유 |
| **커피 추출 향** | 추출 과정의 시각적 아름다움 | MANO 머신의 추출 영상 (스틸컷 or 짧은 루프), 커피가 내려지는 클로즈업, 스팀/크레마 디테일 |
| **매장 공간감** | 공간에 "있고 싶다"는 욕구 자극 | 고퀄리티 공간 사진 (빛·그림자·소재감 강조), 넓은 시야의 인테리어 샷, 갤러리 섹션 |
| **나무·가죽·메탈 질감** | 소재감이 느껴지는 사진 | 테이블 위 커피잔 클로즈업 (나무결 보이도록), 가구 디테일 샷, 소재 텍스처가 느껴지는 앵글 |
| **매장의 조용한 분위기** | 여백과 페이싱으로 "조용함" 표현 | 넉넉한 여백, 부드러운 트랜지션, 과하지 않은 모션, B&W 톤의 차분함 |

### 콘텐츠 아이디어

| 콘텐츠 | 형태 | 위치 |
|--------|------|------|
| **"오늘의 베이킹"** | 인스타그램 연동 or 수동 업데이트 | 메인 페이지 or 카페 메뉴 페이지 |
| **추출 과정 루프 영상** | 5~10초 무음 비디오 루프 | 히어로 섹션 or 스토리 페이지 |
| **공간 갤러리** | 고퀄 사진 그리드 | Good Days (갤러리) 섹션 |
| **"From Bean to Cup"** | 원두 입고→로스팅→추출 스토리라인 | 스토리 페이지 |
| **계절 메뉴 + 시즌 빵** | 시즌 배너 연동 | 메인 페이지 시즌 섹션 |

### 핵심 원칙

> **"보여주되, 설명하지 않는다."**
>
> "저희 매장은 매일 빵을 굽습니다"보다
> 오븐에서 갓 나온 빵의 클로즈업 사진 한 장이 더 강하다.
>
> 사용자가 **"여기 직접 가봐야겠다"**고 느끼면 성공.
