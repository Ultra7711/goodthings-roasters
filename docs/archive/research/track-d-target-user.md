# Track D — 타겟 유저 리서치

> **분석일:** 2026-04-05
> **목적:** GTR의 핵심 고객층을 정의하고, 세그먼트별 웹사이트 이용 시나리오와 니즈를 구체화한다.

---

## 1. 고객 세그먼트 정의

GTR의 비즈니스 특성(로스터리 카페 + 온라인 원두 판매 + 자체 베이커리)을 기반으로 5개 세그먼트를 정의한다.

### 세그먼트 우선순위

1차 전환 목표가 **매장 방문 유도**이므로, 매장과 연결되는 세그먼트가 우선.

| 우선순위 | 세그먼트 | 매장 연결 | 온라인 구매 | 비중(추정) |
|---------|---------|----------|-----------|----------|
| ★★★ | **카페 단골 / 잠재 방문객** | 직접 | 중간 | 40% |
| ★★★ | **디저트·베이커리 팬** | 직접 | 낮음 | 20% |
| ★★☆ | **홈브루어** | 간접 | 높음 | 20% |
| ★★☆ | **선물 구매자** | 간접 | 높음 | 10% |
| ★☆☆ | **커피 입문자** | 간접 | 중간 | 10% |

## 2. 페르소나 카드

### 페르소나 A: 카페 단골 — "지은" (29세, 직장인)

| 항목 | 내용 |
|------|------|
| **직업** | 근처 직장 다니는 마케터 |
| **커피 습관** | 매일 출근 전 또는 점심 후 카페 방문 |
| **GTR 관계** | 매장 2~3회/주 방문, 인스타그램 팔로우 |
| **동기** | 좋은 공간에서 여유 있게 커피 한 잔 |
| **온라인 니즈** | 신메뉴 확인, 매장 영업시간, 원두 집에서도 마시고 싶을 때 |
| **웹사이트 시나리오** | 인스타 → 웹사이트 → 메뉴 확인 → 매장 방문 → 원두 구매 고려 |
| **핵심 페이지** | 카페 메뉴, 매장 정보, 상품 (원두) |

**지은이에게 중요한 것:**
- 카페 메뉴가 예쁘게 잘 보이는 것
- 새로운 시즌 메뉴/디저트 확인
- "역시 여기 좋다"는 재확인 감정 (충성도 강화)

---

### 페르소나 B: 디저트 팬 — "수빈" (33세, 프리랜서)

| 항목 | 내용 |
|------|------|
| **직업** | 프리랜서 디자이너, 카페에서 작업 |
| **커피 습관** | 커피보다 디저트와 공간이 주 목적 |
| **GTR 관계** | 빵 냄새 때문에 재방문, 인스타 릴스에서 발견 |
| **동기** | 갓 구운 빵, 좋은 공간에서 작업 |
| **온라인 니즈** | 오늘 어떤 빵이 있는지, 매장 분위기 사진 |
| **웹사이트 시나리오** | 인스타 → 웹사이트 → 디저트 메뉴 → "오늘 가야겠다" |
| **핵심 페이지** | 카페 메뉴 (디저트), 갤러리, 매장 정보 |

**수빈이에게 중요한 것:**
- 디저트/베이커리 메뉴 이미지가 매력적인 것 (특히 **버터떡** — 현재 킬러 메뉴, "전국 최고"라는 입소문 형성 중)
- 매장 사진에서 "여기서 작업하고 싶다" 느낌
- SNS에 공유할 만한 비주얼 (버터떡 사진은 바이럴 가능성 높음)

---

### 페르소나 C: 홈브루어 — "민수" (36세, 개발자)

| 항목 | 내용 |
|------|------|
| **직업** | 소프트웨어 개발자, 재택근무 |
| **커피 습관** | 집에서 직접 내려 마심 (드립, 에스프레소) |
| **GTR 관계** | 한 번 방문 후 원두 맛에 반함 → 온라인 재구매 |
| **동기** | 좋은 원두를 편하게 집에서 |
| **온라인 니즈** | 원두 정보(원산지, 로스팅, 풍미), 분쇄 옵션, 정기배송 |
| **웹사이트 시나리오** | 매장 방문 → 웹사이트 원두 탐색 → 구매 → 재구매 → 구독 전환 |
| **핵심 페이지** | 상품 목록, 상품 상세 (풍미 노트), 구독 |

**민수에게 중요한 것:**
- 원두의 상세 정보 (원산지, 프로세스, 풍미 프로파일)
- 로스팅 레벨 시각화 (라이트~다크)
- 분쇄 옵션 선택 (홀빈, 드립, 에스프레소)
- 정기배송 vs 일반 구매 가격 비교
- 브루잉 가이드/추천

---

### 페르소나 D: 선물 구매자 — "정원" (31세, 직장인)

| 항목 | 내용 |
|------|------|
| **직업** | 금융권 직장인 |
| **커피 습관** | 커피를 좋아하지만 전문적이진 않음 |
| **GTR 관계** | 지인 추천 or 검색으로 발견 |
| **동기** | 센스 있는 커피 선물을 보내고 싶음 |
| **온라인 니즈** | 선물 세트, 가격대별 필터, 포장, 배송 |
| **웹사이트 시나리오** | 검색/추천 → 웹사이트 → 선물 세트 탐색 → 구매 → 이탈 (1회성) |
| **핵심 페이지** | 상품 (선물 세트), 브랜드 스토리 (선물 가치 확인) |

**정원이에게 중요한 것:**
- "이거 선물하면 센스 있어 보이겠다" 느낌
- 브랜드가 괜찮은 곳인지 빠르게 판단 (웹사이트 퀄리티 = 브랜드 신뢰)
- 가격대별 선물 옵션
- 포장이 예쁜지

---

### 페르소나 E: 커피 입문자 — "하은" (25세, 대학원생)

| 항목 | 내용 |
|------|------|
| **직업** | 대학원생 |
| **커피 습관** | 아메리카노 위주, 스페셜티는 처음 |
| **GTR 관계** | 친구 따라 방문 or SNS에서 발견 |
| **동기** | "스페셜티 커피가 뭐가 다른 거지?" 호기심 |
| **온라인 니즈** | 쉬운 설명, 추천, 입문 세트 |
| **웹사이트 시나리오** | SNS → 웹사이트 → 스토리(브랜드 이해) → 메뉴 확인 → 매장 방문 |
| **핵심 페이지** | 스토리, 카페 메뉴, 매장 정보 |

**하은이에게 중요한 것:**
- 어렵지 않은 진입 (전문용어 벽 X)
- 웹사이트가 예쁘면 "가보고 싶다" 감정 발생
- 브랜드 스토리가 공감되는 것

## 3. 사용자 여정 지도 (User Journey Map)

### 공통 진입 경로

```
                    ┌─ 인스타그램 (가장 활발한 채널)
                    │
진입 ───────────────┼─ 지인 추천 / 입소문
                    │
                    ├─ 네이버 검색 ("OO동 카페", "스페셜티 커피")
                    │
                    └─ 매장 직접 방문 후 웹사이트 재방문
```

### 페르소나별 여정

#### 카페 단골 "지은" — 충성도 강화 루프

```
매장 방문 → [좋은 경험] → 인스타 팔로우
    ↓                         ↓
재방문 ← 웹사이트 ← 인스타 신메뉴 포스팅
    │         │
    │         ├─ 카페 메뉴 확인
    │         ├─ 갤러리 ("역시 좋다")
    │         └─ 원두 구매 시도
    │                │
    └────────────────┘ (선순환)
```

#### 홈브루어 "민수" — 구매 전환 루프

```
매장 방문 → 원두 맛에 반함 → 웹사이트 접속
                                  │
                            원두 탐색 (풍미 노트, 로스팅)
                                  │
                            분쇄 옵션 선택 → 구매
                                  │
                            [만족] → 재구매
                                  │
                            정기배송 비교 → 구독 전환
```

#### 선물 구매자 "정원" — 1회성 전환

```
검색/추천 → 웹사이트 첫 방문
                │
          브랜드 인상 확인 (3초 판단)
          "이 웹사이트 괜찮다 = 브랜드 괜찮다"
                │
          선물 세트 탐색 → 구매 → 이탈
                │
          [받는 사람 만족] → 재유입 가능
```

## 4. 세그먼트별 웹사이트 핵심 니즈 정리

| 니즈 | 지은 (단골) | 수빈 (디저트) | 민수 (홈브루어) | 정원 (선물) | 하은 (입문) |
|------|-----------|-------------|-------------|-----------|-----------|
| 카페 메뉴 | ★★★ | ★★★ | ★☆☆ | ☆☆☆ | ★★☆ |
| 상품 상세 (원두) | ★★☆ | ☆☆☆ | ★★★ | ★★☆ | ★☆☆ |
| 풍미 노트 시각화 | ★☆☆ | ☆☆☆ | ★★★ | ☆☆☆ | ★☆☆ |
| 구독/정기배송 | ★☆☆ | ☆☆☆ | ★★★ | ☆☆☆ | ☆☆☆ |
| 매장 위치/시간 | ★★★ | ★★★ | ★☆☆ | ☆☆☆ | ★★★ |
| 브랜드 스토리 | ★☆☆ | ★☆☆ | ★☆☆ | ★★☆ | ★★★ |
| 갤러리/공간 사진 | ★★☆ | ★★★ | ☆☆☆ | ★☆☆ | ★★★ |
| 선물 세트 | ☆☆☆ | ☆☆☆ | ☆☆☆ | ★★★ | ☆☆☆ |
| 시즌 메뉴/이벤트 | ★★★ | ★★★ | ★☆☆ | ☆☆☆ | ★★☆ |
| SNS 연동 | ★★★ | ★★★ | ★☆☆ | ☆☆☆ | ★★★ |

### 핵심 인사이트

**① 카페 메뉴가 가장 많은 세그먼트에 영향**
단골 + 디저트팬 + 입문자 = 전체의 ~70%. 모두 카페 메뉴를 필요로 함.
→ **카페 메뉴 페이지의 퀄리티가 웹사이트 전체 만족도를 좌우.**
→ 특히 **버터떡**은 현재 가장 강력한 매장 방문 트리거. "전국 최고"라는 입소문이 이미 있으므로, 메뉴 페이지에서 가장 매력적으로 보여야 할 아이템. 시즌 배너·어나운스 영역에서도 활용 가능.

**② 풍미 노트 시각화는 홈브루어 전용이지만 차별화 가치 극대**
전체 비중 ~20%에 불과하지만, 이 세그먼트가 **재구매 + 구독 전환의 핵심**.
풍미 그래픽은 "이 브랜드 진짜 신경 쓴다"는 인상을 다른 세그먼트에게도 전달.

**③ 브랜드 스토리는 첫 방문자에게 집중**
단골에게는 불필요하지만, 선물 구매자와 입문자에게는 **신뢰 형성의 핵심**.
→ 메인 페이지에서 가볍게 노출 + 서브 페이지에서 깊이.

**④ SNS ↔ 웹사이트 연결이 진입의 핵심 축**
인스타그램이 가장 활발한 채널. 인스타 → 웹사이트 → 매장의 흐름이 핵심.
→ 푸터·메인 페이지에 인스타 연동 유지, 시즌/신메뉴를 두 채널에서 동시 노출.

## 5. 웹사이트 정보 구조(IA) 제안

세그먼트별 니즈와 전환 우선순위를 반영한 메인 페이지 섹션 순서:

```
1. 히어로 영상 ─────── 모든 세그먼트: 첫 인상, 브랜드 경험
2. 시즌 배너 ──────── 단골 + 디저트팬: 새로운 소식
3. 카페 메뉴 카드 ──── 단골 + 디저트팬 + 입문자: 매장 방문 유도 (1차 전환)
4. 브랜드 스토리 ──── 입문자 + 선물: 신뢰 형성
5. 상품 스크롤 ────── 홈브루어 + 선물: 원두 구매 (2차 전환)
6. 서비스 카드 ────── 홈브루어: 구독 전환 (3차 전환)
7. Come visit us ──── 모든 세그먼트: 매장 방문 CTA (최종 전환)
8. 갤러리 ──────── 디저트팬 + 입문자: 공간 매력
9. 푸터 (SNS) ────── 모든 세그먼트: 인스타 연결
```

> 현재 GTR 프로토타입의 섹션 순서와 거의 일치. 리서치 결과가 기존 설계를 검증한 셈.
