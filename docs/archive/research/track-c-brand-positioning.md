# Track C — 브랜드 포지셔닝

> **분석일:** 2026-04-05
> **목적:** GTR의 브랜드 포지션을 경쟁사 대비 명확히 정의하고, 웹사이트가 전달해야 할 핵심 브랜드 메시지를 도출한다.

## 1. 포지셔닝 맵

### 축 정의
- **X축:** 전통/클래식 ← → 모던/트렌디
- **Y축:** 대중적/접근성 ← → 프리미엄/고급

### 배치

```
                        프리미엄/고급
                             │
                             │
               Onyx ●        │         ● % Arabica
                             │
                             │    ◆ GTR (목표)
                             │       ● Onibus
                             │
   전통/클래식 ──────────────┼──────────────── 모던/트렌디
                             │
                             │
              Verve ●        │
                             │
                  Blue Bottle ●
                             │
                             │
                        대중적/접근성
```

### 포지셔닝 해석

| 브랜드 | X축 (스타일) | Y축 (가격대/포지션) | 키워드 |
|--------|------------|-------------------|--------|
| **Onyx Coffee Lab** | 모던 쪽이지만 에디토리얼 감성 | 최상위 프리미엄 | 럭셔리, 에디토리얼, 수상, 크래프트 |
| **% Arabica** | 극단적 모던/미니멀 | 프리미엄 | 교토, 극미니멀, 영화적, 코스모폴리탄 |
| **Onibus Coffee** | 모던 (일본식 정갈함) | 프리미엄~중상 | 정갈, 깔끔, 커넥팅, 일본 감성 |
| **Blue Bottle** | 모던~중립 | 중상 (대중적 프리미엄) | 클린, 접근성, 쇼핑 최적화 |
| **Verve Coffee** | 전통~중립 (크래프트 감성) | 중상 | 아티자날, 따뜻함, 라이프스타일 |
| **◆ GTR** | **모던/트렌디** | **프리미엄~중상** | 모던, 정갈, 따뜻함, B&W |

### GTR 포지션 정의

> **"모던한 정갈함 속의 따뜻함"**
>
> % Arabica의 B&W 미니멀에서 한 걸음 내려와 인간미를 더한 포지션.
> Onibus의 일본식 정갈함에서 한 걸음 올라가 프리미엄을 더한 포지션.
> 차갑지도, 과하지도 않은 **"단정한 따뜻함"**.

## 2. 브랜드 속성 정의서

### 핵심 정보

| 항목 | 내용 |
|------|------|
| **브랜드명** | Good Things Roasters (GTR) |
| **슬로건** | "good things, take time" |
| **핵심 가치** | 좋은 원두 + 좋은 추출(MANO) + 좋은 베이커리 |
| **1차 전환 목표** | 매장 방문 유도 → 브랜드 충성도 형성 |
| **톤앤무드** | 모던·정갈한 B&W + 따뜻함 |

### 브랜드 성격 스펙트럼

각 항목에서 GTR이 어디에 위치하는지:

```
차가운 ──────────●───── 따뜻한         → 중간~따뜻한 (B&W지만 인간미)
미니멀 ─────────────●── 풍부한         → 미니멀 쪽 (깔끔함 우선)
빠른   ─────────────●── 느린           → 느린 쪽 ("take time")
공격적 ──────────────●─ 절제된         → 절제된 쪽 (급박한 톤 지양)
젊은   ────────●──────── 성숙한         → 중간~성숙 (트렌디하지만 가벼운 건 아님)
대중적 ─────────●─────── 전문적         → 중간 (스페셜티지만 입문자도 환영)
```

### 브랜드 보이스 & 톤

| 항목 | 방향 | 예시 |
|------|------|------|
| **어조** | 차분하고 자신감 있는, 과시하지 않는 | "좋은 원두를 씁니다" (O) / "최고급 프리미엄 원두!" (X) |
| **문장 길이** | 짧고 여운이 있는 | "천천히 내린 한 잔." |
| **영문** | 소문자 선호, 간결 | "good things, take time" (올캡 아님) |
| **한글** | 존댓말 기반, 부드러운 | "만나보세요" (O) / "만나봐!" (X) |
| **피해야 할 톤** | 급박, 과장, 힙스터 과잉 | "지금 바로!", "한정 수량!", "갓생" |

## 3. 경쟁사 대비 차별화 포인트

### GTR만의 것

| 차별점 | 설명 | 웹사이트 표현 |
|--------|------|-------------|
| **MANO 머신** | 일반 에스프레소 머신과 확실히 다른 미래적 추출 장비 | 히어로 영상(1차), 스토리 페이지(2차), 장비 소개 |
| **자체 베이커리** | 매장에서 직접 굽는 빵·디저트 — 갓 구운 냄새가 공간에 머무는 경험. **킬러 메뉴: 버터떡** — "전국 어느 카페·베이커리보다 맛있다"는 입소문 형성 중 | 카페 메뉴 이미지, SNS 연동, 매장 방문 유도 콘텐츠. 버터떡은 시즌 배너·어나운스·메뉴 상단 노출로 방문 트리거 활용 |
| **총체적 공간 경험** | 커피 + 베이킹 향 + 화이트·메탈·우드 인테리어 | 공간 갤러리, 히어로 영상, "Come visit us" |
| **온라인 카페 메뉴** | 경쟁사 아무도 구현하지 않는 영역 | 이미지 중심 메뉴 페이지 — GTR 독자 콘텐츠 |
| **풍미 노트 시각화** | 경쟁사 모두 텍스트 나열 수준 | 인터랙티브 풍미 그래픽 — 업계 최초 수준 |

### 경쟁사와 공유하는 것 (기본 자격)

| 항목 | 수준 |
|------|------|
| 스페셜티 등급 원두 | 동등 |
| 직접 로스팅 | 동등 |
| 구독/정기배송 | 구현 예정 |
| 깔끔한 웹사이트 | 목표: 최상위 수준 |

## 4. 웹사이트 브랜드 메시지 프레임워크

### 메시지 계층

```
Level 0 — 철학 (웹사이트 전체에 스며드는 결)
  "good things, take time"
  → 직접 말하지 않고, 페이싱·여백·인터랙션으로 체감

Level 1 — 핵심 메시지 (히어로 + 스토리 섹션)
  "좋은 원두, 좋은 추출, 좋은 시간"
  → 히어로 영상으로 압축 전달 (MANO + 로스팅 + 공간)

Level 2 — 섹션별 메시지
  카페 메뉴 → "오늘, 매장에서 만나보세요"
  상품      → "집에서도 같은 맛을"
  구독      → "매달, 좋은 원두를 문 앞에"
  스토리    → "시간을 들여 만듭니다"
  갤러리    → "이런 공간에서"

Level 3 — 상세 메시지 (서브 페이지, 상품 상세)
  원두별 원산지·프로세스·풍미 스토리
  MANO 머신 소개
  베이커리 철학
```

### 전환 목표별 메시지 우선순위

| 전환 목표 | 메시지 | 섹션/위치 |
|----------|--------|----------|
| **매장 방문** (1순위) | "직접 오셔서 느껴보세요" | 카페 메뉴, Come visit us, 갤러리 |
| **브랜드 인지** (2순위) | "좋은 것엔 시간이 필요합니다" | 히어로, 스토리 |
| **원두 구매** (3순위) | "집에서도 같은 맛을" | 상품 섹션, 상품 상세 |
| **구독 전환** (4순위) | "매달, 문 앞까지" | 서비스 카드, 상품 상세 토글 |

## 5. 비주얼 아이덴티티 — 웹사이트 적용

### 컬러 시스템

| 용도 | 컬러 | 비고 |
|------|------|------|
| **배경** | #FFFFFF (화이트) | 주 배경 |
| **보조 배경** | #F5F5F5 ~ #FAFAFA | 섹션 구분, 카드 배경 |
| **텍스트 (주)** | #1A1A1A | 헤드라인, 본문 |
| **텍스트 (보조)** | #666666 | 설명문, 캡션 |
| **텍스트 (약)** | #999999 | 힌트, 비활성 |
| **라인/구분선** | rgba(0,0,0,.06~.10) | 섹션 구분, 카드 보더 |
| **CTA 버튼** | #1A1A1A (bg) + #FFFFFF (text) | 솔리드 버튼 |
| **CTA 호버** | #333333 | 버튼 호버 |
| **액센트** | 최소 사용 — 이미지 색감에 의존 | % Arabica 접근 참고 |

### 이미지 톤 가이드

| 항목 | 방향 |
|------|------|
| **조명** | 자연광 기반, 부드러운 그림자 |
| **색온도** | 약간 따뜻한 중립 (차가운 형광등 느낌 X) |
| **채도** | 낮은~중간 — HDR 느낌 X |
| **배경** | 실제 매장 공간 또는 깨끗한 단색 |
| **피사체** | 소재감이 느껴지는 앵글 (나무결, 메탈 표면, 크레마 디테일) |
| **인물** | 최소 노출 — 공간과 제품 중심 |
