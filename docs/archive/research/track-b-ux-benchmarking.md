### 핵심 발견

**① 풍미 노트는 모두 "부족"**
어떤 브랜드도 상품 리스팅 카드에서 풍미 노트를 시각적으로 잘 보여주지 못함. Onyx가 상세 페이지에서 텍스트로 나열하는 수준이 최상위. **GTR이 카드 또는 상세 페이지에서 인터랙티브 풍미 그래픽을 구현하면 업계 최초 수준의 차별화.**

**② 분쇄 옵션 = 전환 촉진제**
Onibus, Verve 모두 4가지 분쇄 옵션 제공. 홈브루어에게 "나에게 맞는 분쇄"를 바로 선택할 수 있게 하는 것이 구매 허들을 낮춤.

**③ 필터가 없는 브랜드가 많음**
Onibus는 필터/정렬 없이 정적 리스팅. 상품 수가 적으면 가능하지만, GTR도 초기 상품 수가 적다면 필터 없이 깔끔한 그리드로 충분.

## 플로우 2. 구독/정기배송

### 비교 매트릭스

| 항목 | Onyx Coffee Lab | Blue Bottle | Onibus Coffee | Verve Coffee |
|------|----------------|-------------|---------------|-------------|
| **플랜 수** | 3종 (Roaster's Choice, Blend Cycle, Southern Weather) | 구독 + 멤버십 분리 | 1종 (싱글 오리진 월간) | 다수 (지역별: Africa, Latin America, Espresso) |
| **가격** | $21.50~$164 (사이즈별) | 중상 | ¥4,200/월 (150g×2) | $26~ |
| **배송 주기** | 1/2/3/4주 선택 | 미확인 | 월간 고정 | 다양 |
| **무료 배송** | $40 이상 무료 | 프로모 연동 | ✅ 구독 무료 배송 | ✅ 구독 무조건 무료 |
| **구독 할인** | ~5% (블렌드) | 미확인 | 무료 배송이 실질 할인 | 구독 가격 별도 표시 |
| **가치 제안 방식** | 에디토리얼 카피 (연 70~100종) | 아이콘 기반 설명 | 간단한 설명 | 지역별 큐레이션 강조 |
| **선물 구독** | ✅ | ✅ | 미확인 | ✅ |

### 핵심 발견

**① Onyx의 가치 제안 카피가 독보적**
"연간 70~100가지 싱글 오리진"이라는 숫자가 구독의 가치를 명확히 전달. 감성적 카피 + 구체적 숫자 조합.

**② 구독 vs 일반 구매 "즉시 비교"는 아무도 잘 안 함**
GTR이 원하는 "상품 상세에서 일반 vs 정기배송 가격/혜택 즉시 비교"는 경쟁사 중 제대로 구현한 곳이 없음. Verve가 구독 가격을 별도 표시하는 수준. **GTR이 이걸 잘 구현하면 전환율 차별화 가능.**

**③ 무료 배송 = 구독의 가장 강력한 혜택**
Onibus, Verve 모두 구독에 무료 배송 제공. 이것 자체가 구독의 핵심 인센티브.

**④ Onibus의 단순함**
1개 플랜, ¥4,200, 월간 고정 — 선택 피로가 없음. GTR 초기에는 이런 단순한 모델이 적합할 수 있음.

### 비교 매트릭스

| 항목 | Onyx Coffee Lab | Blue Bottle | % Arabica | Onibus Coffee | Verve Coffee |
|------|----------------|-------------|-----------|---------------|-------------|
| **메뉴 페이지** | Locations에서 간접 | 별도 메뉴 페이지 없음 (매장 찾기 중심) | Locations만 | 별도 메뉴 페이지 없음 | Locations에서 간접 |
| **메뉴 이미지** | ❌ | ❌ | ❌ | ❌ | ❌ |
| **카테고리 구분** | — | — | — | — | — |
| **매장 연결** | 위치 정보 | 위치 + 지도 | 위치 목록 (도시별) | 위치 정보 | 위치 + 지도 |

### 핵심 발견

> **⚠️ 어떤 브랜드도 온라인 카페 메뉴를 제대로 구현하지 않음.**

모든 경쟁사가 "Locations" 페이지에서 매장 위치만 제공하고, 카페 메뉴(음료·디저트)를 웹사이트에서 보여주는 곳이 사실상 없음.

이유 추정:
- 해외 브랜드는 **온라인 원두 판매**가 주 매출이므로 메뉴 페이지 불필요
- 카페 메뉴는 매장에서 직접 확인하는 것이 당연시됨

**GTR에게 이것은 큰 기회:**
- GTR의 1차 목표가 **매장 방문 유도**이므로 카페 메뉴가 핵심 콘텐츠
- **이미지로 승부하는 카페 메뉴 페이지**는 경쟁사 누구도 하지 않는 것
- 메뉴 사진 자체가 "여기 가보고 싶다"를 만드는 가장 직접적인 도구
- 특히 **자체 베이커리/디저트**는 일반 커피숍과의 차별점이므로 이를 시각적으로 강조하면 매장 방문 전환에 매우 효과적

## 플로우 4. 브랜드 스토리 탐색

### 비교 매트릭스

| 항목 | Onyx Coffee Lab | Blue Bottle | % Arabica | Onibus Coffee | Verve Coffee |
|------|----------------|-------------|-----------|---------------|-------------|
| **스토리 형태** | 에디토리얼 (산지·생산자 스토리) | About 페이지 (미션·가치) | **Philosophy 페이지** + Films | About 페이지 | Story 페이지 |
| **콘텐츠 깊이** | 깊음 — 생산자별 스토리 | 중간 | 깊음 — 필름 콘텐츠 제작 | 중간 | 중간 |
| **미디어** | 이미지 + 텍스트 | 이미지 + 텍스트 | **비디오 중심** (Films 섹션) | 이미지 + 텍스트 | 이미지 + 텍스트 |
| **감성 vs 정보** | 감성 60% + 정보 40% | 정보 중심 | **감성 90%** | 정보 중심 | 균형 |
| **메인에서 진입** | 산지 스포트라이트 섹션 | 최소 노출 | 히어로 비디오가 곧 스토리 | 배너 | CTA 버튼 |

### 핵심 발견

**① % Arabica의 "Films" 접근이 GTR에 가장 참고할 만함**
% Arabica는 텍스트로 설명하지 않고 **영상으로 브랜드를 말함**. GTR의 히어로 영상 접근과 동일 철학. 스토리 페이지에서도 영상/사진 중심 + 최소 텍스트로 가는 것이 "보여주되 설명하지 않는다" 원칙과 일치.

**② Onyx의 생산자 스토리는 깊이의 좋은 예시**
원두별 산지와 생산자 이야기를 에디토리얼 수준으로 다룸. GTR이 원두 상세 페이지에서 이런 깊이를 일부 구현하면 "진심" 전달에 효과적.

**③ 메인 페이지에서 스토리로의 진입**
GTR 현재 구조(스토리 섹션 → "THE STORY" CTA)는 적절. % Arabica처럼 히어로 자체가 스토리 역할을 겸하므로, 별도 섹션은 "더 알고 싶은 사람"을 위한 2차 경로.

## 종합 인사이트 — GTR UX 설계 방향

### 경쟁사가 잘 하는 것 (따라가야 할 것)

| 패턴 | 참고 브랜드 | GTR 적용 |
|------|-----------|---------|
| 분쇄 옵션 4종 제공 | Onibus, Verve | 상품 상세에서 분쇄 옵션 선택 |
| 구독 무료 배송 | Onibus, Verve | 구독 혜택의 핵심 인센티브로 활용 |
| 에디토리얼 수준의 카피 | Onyx | 원두 설명, 구독 가치 제안에 적용 |
| 영상 중심 스토리텔링 | % Arabica | 히어로 영상 + 스토리 페이지 |

### 경쟁사가 못 하는 것 (차별화 기회)

| 미개척 영역 | 경쟁사 현황 | GTR 기회 |
|-----------|-----------|---------|
| **풍미 노트 시각화** | 모두 텍스트 나열 또는 미표시 | 인터랙티브 풍미 그래픽 (로스팅 레벨 게이지, 플레이버 휠, 노트 아이콘) |
| **구독 vs 일반 구매 즉시 비교** | 아무도 제대로 미구현 | 상품 상세에서 가격/혜택 토글 비교 |
| **온라인 카페 메뉴** | 사실상 전무 | 이미지 중심 카페 메뉴 + 카테고리 구분 + 매장 방문 유도 |
| **베이커리/디저트 콘텐츠** | 해당 없음 | GTR만의 차별점 — 갓 구운 빵·디저트 비주얼로 매장 방문 전환 |

### GTR UX 설계 원칙 (초안)

1. **"보여주되, 설명하지 않는다"** — 이미지와 영상이 텍스트보다 먼저
2. **여유로운 페이싱** — 급하지 않은 스크롤, 넉넉한 여백, 부드러운 트랜지션
3. **감각의 트리거** — 온라인에서 직접 전달 못하는 감각(냄새, 질감)을 시각으로 환기
4. **정보의 계층화** — 관심 있는 사람만 더 깊이 들어갈 수 있는 구조 (카드 → 상세)
5. **구매 허들 최소화** — 분쇄 옵션, 구독 비교, 빠른 장바구니
6. **매장 방문 = 궁극의 CTA** — 모든 경로가 "여기 가보고 싶다"로 수렴
