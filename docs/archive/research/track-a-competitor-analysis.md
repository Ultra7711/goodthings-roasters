# Track A — 경쟁사 벤치마킹 분석 (상위 5개)

> **분석일:** 2026-04-05
> **분석 대상:** GTR 목표에 가장 근접한 해외 스페셜티 커피 브랜드 5개
> **분석 기준:** 비주얼, 레이아웃, 히어로, 인터랙션, 상품 표시, 구독 모델
>
> **참고:** 국내 스페셜티 브랜드(프릳츠, 펠트, 모모스 등)는 대부분 카페24/Shopify 등 플랫폼 템플릿 한계 안에서 타협한 수준이며, GTR이 목표하는 개발 스펙에 미달. 벤치마크 대상으로 부적합하여 **해외 상위 브랜드 중심으로 분석**한다. 국내 시장에서 해외 수준의 웹사이트 퀄리티를 가진 스페셜티 브랜드가 사실상 부재하므로, GTR이 이를 달성하면 국내 시장 자체에서 독보적 차별화가 가능하다.

## 종합 인사이트

### GTR에 적용할 핵심 배움 5가지

#### 1. B&W 클린 + 따뜻함의 균형
% Arabica의 B&W 순수함은 GTR 톤과 맞지만, 너무 차갑고 건조함.
Onibus의 정갈한 따뜻함을 섞어 **"모던하지만 인간미 있는 B&W"** 를 만들어야 함.
→ 방법: 이미지 톤에서 따뜻함을 주고, 레이아웃/타이포는 차갑게 유지.

#### 2. 히어로 영상은 최강 무기 — 단, 편집 품질이 전부
% Arabica의 풀스크린 비디오 접근이 GTR과 가장 유사.
차이점: GTR은 3가지 요소(MANO + 로스팅 + 공간)를 한 영상에 압축.
→ **씬 전환의 호흡, 색보정의 일관성, 음악/무음 선택**이 웹사이트 전체 인상을 결정.

#### 3. 풍미 노트 그래픽 = 미개척 차별화 영역
모든 경쟁사가 텍스트 나열 또는 최소 표시에 머무름.
GTR이 **인터랙티브한 풍미 그래픽(로스팅 레벨 시각화, 플레이버 휠, 노트 아이콘)**을 구현하면 업계 최초 수준의 차별화.

#### 4. 메인 페이지 섹션 순서 재고
매장 방문 유도가 1차 목표이므로 현재 순서(카페 메뉴 → 스토리)는 합리적.
다만 경쟁사들은 상품을 매우 일찍 노출. **시즌 배너에 카페 메뉴 연결**을 넣거나, 상품과 메뉴의 경계를 유연하게 설계하는 방안 고려.

#### 5. 인터랙션 수준은 이미 경쟁력 있음
GTR 프로토타입의 blur→sharp, fade-in stagger, 파티클 등은 Onyx에 근접.
과하지 않게 통제만 잘 하면 "기분 좋은 인터랙션" 목표 달성 가능.
