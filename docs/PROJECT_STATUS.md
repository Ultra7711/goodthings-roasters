## 1. 한눈에 보는 요약

Good Things Roasters 는 스페셜티 커피 로스터리의 자사몰과 카페 브랜드 사이트를 통합한 웹 서비스입니다. 외부 솔루션(카페24·스마트스토어) 대신 자체 코드로 구축하여 결제 수수료·플랫폼 수수료를 절감하고, 브랜드 톤을 완전히 통제하는 것을 목표로 합니다.

| 영역 | 진행률 | 요지 |
|---|---|---|
| **디자인** | 100% | 디자인 시스템·프로토타입·반응형 4단계 모두 확정 |
| **프론트엔드 (고객 화면)** | 약 98% | 메인·샵·상품 상세·장바구니·결제·마이페이지·검색·매장 정보 모두 작동 |
| **백엔드 (서버·DB)** | 약 92% | 회원·주문·결제·정기배송 엔진 모두 운영 가능. 정기배송 자동 결제 cron만 미구축 |
| **어드민 (운영자 화면)** | 약 85% | 주문·회원·사용자·정기배송·통계·사이트설정·갤러리·카페 이벤트 운영 가능. 상품 등록 폼 일부와 카페 메뉴 운영 UI 잔여 |
| **인프라·배포** | 약 90% | Vercel·Supabase·Sentry·Resend 모두 가동. 결제 라이브 키 전환과 운영 SOP 문서화가 잔여 |

**출시 준비도:** 핵심 거래 흐름(상품 탐색 → 결제 → 주문 확인 → 배송 추적 → 정기배송)이 모두 작동하므로, 카탈로그 데이터 입력과 결제 라이브 키 전환만 끝나면 일반 고객 오픈이 가능한 상태입니다. 정기배송 자동 결제 cron 은 출시 이후 첫 결제 사이클 도래 전(약 1개월 내)에 보완 예정입니다.

### 4.2 사용 기술

| 영역 | 기술 | 선택 이유 |
|---|---|---|
| **웹 프레임워크** | Next.js (최신 16) + React 19 | 한 코드베이스로 고객·어드민 동시 구축. 서버 렌더링으로 검색엔진 노출 유리 |
| **UI 라이브러리** | Radix UI (headless) + 자체 디자인 토큰 | 접근성·키보드 네비게이션 기본 제공. 디자인은 100% 자체 통제 |
| **스타일** | Tailwind CSS v4 + CSS Custom Properties | 빠른 개발 + 디자인 시스템 토큰화 |
| **DB·인증·스토리지** | Supabase (PostgreSQL 기반) | 데이터베이스·회원 인증·이미지 저장소 통합. 자체 서버 운영 부담 없음 |
| **결제** | 토스페이먼츠 직접 연동 | 카페24 솔루션 대비 수수료 절감 (실질 합산 1.5~3.5% vs 5.0~6.5%) |
| **이메일** | Resend (발신) + Google Workspace (수신) | 자동 발신 무료 한도 3,000건/월 + 도메인 이메일 별도 |
| **호스팅** | Vercel | Next.js 공식 플랫폼 · GitHub push 자동 배포 · HTTPS 자동 |
| **모니터링** | Sentry + Vercel Analytics + Speed Insights | 에러·성능·사용자 행동 통합 추적 |
| **폼·검증** | React Hook Form + Zod | 클라이언트·서버 동일 스키마 사용 |
| **데이터 페치** | TanStack Query | 캐싱·낙관적 업데이트·자동 재시도 |
| **드래그 정렬** | dnd-kit | 어드민 이미지·옵션 행 재정렬 |

### 4.3 보안 설계

| 위협 | 대응 |
|---|---|
| 데이터 무단 접근 | Row Level Security 모든 테이블 적용 — 본인 데이터만 조회 가능 |
| 결제 위조 | 토스페이먼츠 웹훅 하이브리드 인증 (카드: GET 재조회, 가상계좌: timing-safe secret) |
| XSS·CSRF | per-request nonce Content Security Policy + HttpOnly 쿠키 |
| 무차별 대입 공격 | Upstash Redis 기반 Rate Limit |
| 비밀번호 유출 | Argon2 해싱 |
| 운영자 권한 오남용 | Owner / Staff 2단계 권한 + 모든 액션 감사 로그 |

### 5.3 강점 요약

1. **자체 결제 직연동** — 카페24·플랫폼 솔루션 대비 결제·플랫폼 수수료 약 2~3%p 절감
2. **운영자 자유도** — 시그니처 챕터·카페 이벤트·시즌 배너·약관 모두 운영자가 어드민에서 직접 편집 가능 (개발자 개입 불필요)
3. **권한 분리·감사 로그** — Owner/Staff 권한 + 모든 액션 추적으로 운영 사고 대비
4. **현대적 기술 스택** — Next.js 16·React 19·Supabase 등 최신 안정 버전 채택. 향후 5년 이상 유지보수 가능
5. **반응형 4단계 완성** — PC·랩탑·태블릿·모바일 모든 화면에서 검증된 디자인
6. **모니터링 풀 셋업** — Sentry·Vercel Analytics 로 출시 즉시 실시간 장애·성능 추적
