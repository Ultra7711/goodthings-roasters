## 자문 1 프롬프트 (복사용)

```
안녕하세요. Specialty coffee 로스터리 B2C 웹사이트 "Good Things Roasters (굳띵즈)"
의 출시 전 V2 리뉴얼 자문을 받고 싶습니다.

## 컨텍스트

- Next.js 16 + Tailwind v4 + Radix 기반 SPA-like 사이트, 13 페이지.
- 현재 디자인 톤: Warm-shifted Black & White, editorial, 잔잔한 여백, 한국어 본문 우선.
- 출시 전 단계 — 시간 여유 있음. 작은 폴리싱이 아닌 V2 큰 전환 방향을 잡는 것이 목표.
- 첨부한 `design-audit-input.md` 에 브랜드 톤, 페이지 인벤토리 (13), 핵심 컴포넌트,
  디자인 토큰, 알려진 UX 통점, 벤치마크 (Blue Bottle ~50% · Drop Coffee ~30% · Aesop /
  MUJI / Apple ~20%), 제약 조건이 정리되어 있습니다.
- 첨부 스크린샷 (페이지별 desktop 1440 + mobile 360) 으로 현재 디자인을 확인할 수
  있습니다.

## 자문 범위

전체 13 페이지를 대상으로 하되, **메인 (`/`) 과 마이페이지 (`/mypage`) 를 집중 자문**
하고 싶습니다. 나머지 11 페이지는 시스템 가이드 적용 + UX 우선순위 + 컴포넌트 단위
개선안으로.

## 깊이

1. **디자인 방향 자문** — 현재 톤 유지/전환 판단, 큰 그림 톤 시프트 권고 (전체)
2. **컴포넌트 단위 개선안** — 헤더·카드·아코디언·드로어·인풋 등 (전체)
3. **픽셀 단위 시안** — 메인 + 마이페이지에 한정. 데스크탑 1440 기준, 핵심 섹션별
   레이아웃·타이포·여백·인터랙션 묘사 (Figma 파일 생성 불필요 — 텍스트 + 컴포넌트
   설명 + ASCII/마크다운 와이어프레임으로 충분)

## 출력 형식 (순서대로 출력)

### 1. 통합 디자인 시스템 가이드 (전체 적용)
- 색상 팔레트 — 현재 토큰 평가 + 신규/변경 제안
- 타이포 스케일 — 현재 vs 권고
- 그리드·여백 — 1440 데스크탑 + 360 모바일 권고
- 컴포넌트 패턴 — 표준화·일관성 측면 권고
- 모션·인터랙션 톤 — Blue Bottle/Drop Coffee 풍 적용 정도

### 2. 페이지별 개별 시안 (메인 + 마이페이지)
- 섹션별 레이아웃 (와이어프레임 + 설명)
- 핵심 컴포넌트 적용
- 인터랙션·모션 권고
- 모바일 적용 노트

### 3. UX 우선순위 리스트 (전체 13 페이지)
- 페이지별 상위 3 개 개선 항목
- 우선순위 (🔴 critical / 🟠 high / 🟡 medium / 🟢 low)
- 각 항목당 권고 액션 1~2 줄
- 알려진 통점 (`design-audit-input.md` §5) 매칭 표시

## 시점 / 톤

- **출시 전 V2 큰 전환** — 작은 폴리싱이 아닌 방향성 의사결정.
- 단, 브랜드 톤 (Warm-shifted B&W, Pretendard 단일) 은 유지. 색조 큰 변경 자제.
- 한국어 본문 (정중한 ~요/~습니다 체) 정연한 정보 톤 유지.
- "less is more" 표본 — Blue Bottle 식 photography forward 적용 가능성 검토.

## 제약·금지

- ❌ Figma 파일 생성 (텍스트 + 와이어프레임만)
- ❌ Fraunces 등 serif 폰트 도입 (Pretendard 단일 통일 결정 완료)
- ❌ 한국 specialty coffee 브랜드 (프릳츠·모모스 등) 직접 모방
- ❌ 화려한 그라데이션 / 네온 / 인터럽티브 챗봇 모달
- ❌ shadcn/ui 도입 (기본 Radix headless + 자체 토큰 시스템 유지)
- ✅ Blue Bottle / Drop Coffee 톤 적극 차용
- ✅ 미니멀·여백·photography forward
- ✅ Pretendard 활용한 한국어 타이포 강화
```
