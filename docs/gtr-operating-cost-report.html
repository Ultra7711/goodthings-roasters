3. 이전 보고서(v2.0·2026-04) 정정 사항

⚠️ v2.0 는 그대로 사용 금지 — 아래가 정정됨
  • Vercel "런칭 초기 Hobby 무료 ₩0" → ❌ 오류. 결제·판매 사이트는 Hobby 약관 위반(상업 사용 금지)이라 런칭부터 Pro($20/월)가 필수. "초기 ₩0"은 성립하지 않음.
  • 원화 총액 표기(월 약 ₩10,500 등) → 환율·플랜 변동으로 단정 부적절 → 공식 USD 기준으로 대체, 총액 단정 제거.
  • Upstash·Sentry 누락 → 본 보고서에 추가(둘 다 현재 ₩0).

4. 데이터 백업 (이번 세션 조치)

✅ 무료 자동 백업 구축 완료
Supabase 무료 티어는 관리형 백업이 없으나, GitHub Actions로 매일 자동 덤프 → gpg(AES256) 암호화 → 보관하도록 구성했다. 추가 비용 없이 결제·주문 데이터 백업을 확보 → 백업 목적의 Supabase Pro 결제는 불필요.

5. 확인이 필요한 항목 (미검증)

  • 토스페이먼츠 결제 수수료율 — 가맹 계약·정산 조건
  • 도메인 등록처·연 갱신비
  • Google Workspace 사용 여부·요금
  • 각 무료 티어의 초과 시점 — 실제 트래픽·데이터 누적 후 대시보드 확인

6. 결론

  • 강제 월 고정비 = Vercel Pro $20부터(USD). 그 외 인프라는 우리 규모에서 무료로 시작 가능.
  • 깜짝 청구 위험 낮음 — 무료 티어형(Upstash·Sentry 등)은 카드 미등록 또는 초과 시 중단 구조라 자동 청구가 없음. 실청구는 Vercel(구독)과 토스 수수료(매출 연동)에 한정.
  • 데이터 백업은 무료로 해결(Supabase Pro 불필요).
  • 총액·원화 환산은 본 보고서에서 단정하지 않음 — 미검증 항목(§5) 확정 후 합산 권장.
