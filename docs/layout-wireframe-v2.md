## 공통 요소

### Announcement Bar
- 텍스트: `30,000원 이상 구매 시 무료 배송`
- 배경: `#1C1B19` (black) / 텍스트: `#FAFAF8` (bg)
- 높이: 36px / 폰트: 13px (`--type-body-s-size`) weight 400

### Header (글래스모피즘)
- 로고: `good things` (Inter 600, lowercase)
- 네비게이션: `The Story` · `Menu` · `Shop` · `Good Days`
- 아이콘: 검색 · 계정 · 장바구니
- 높이: 60px
- **라이트:** `rgba(250, 250, 248, .72)` + `backdrop-filter: blur(16px)` — `#FAFAF8` 72%
- **다크:** `rgba(28, 27, 25, .75)` + `backdrop-filter: blur(16px)` — `#1C1B19` 75%
- 스크롤 위치의 섹션 `data-header-theme` 값에 따라 라이트 ↔ 다크 동적 전환
- 전환 트랜지션: `700ms ease`

### 서브 페이지 헤더 (글래스모피즘)
- `← 페이지명` + 우측 아이콘
- 높이: 60px
- **라이트:** `rgba(250, 250, 248, .72)` + `backdrop-filter: blur(16px)` — 메인 헤더와 동일
- **다크:** `rgba(28, 27, 25, .75)` + `backdrop-filter: blur(16px)` — 메인 헤더와 동일
- 서브 페이지도 섹션별 `data-header-theme`으로 스크롤 기반 테마 전환 지원
- 장바구니 헤더만 예외: `rgba(250, 250, 248, .92)` (92% 불투명)

### 구조

1. **어나운스 바** + **서브 헤더** (`← CAFE MENU`)
2. **필터 탭**: 전체 · 시그니처 · 커피 · 티 · 음료 · 디저트
3. **메뉴 그리드**: `repeat(3, 1fr)` gap: `clamp(12px, 1.4vw, 20px)`, 비율 1:1
4. **하단 CTA**: `매장에서 직접 만나보세요.` → `매장 정보 보기`
5. **사이드 패널 (영양 정보)**
   - 우측 슬라이드 인, 너비: 540px
   - `영양 정보` 타이틀 + 닫기 버튼
   - `1회 제공량 355ml 기준`
   - 3컬럼 그리드: 칼로리, 포화지방, 당류, 나트륨, 단백질, 카페인

### 매거진 그리드 — 5종 패턴 순환

이미지는 A→B→C→D→E 패턴을 순환하며 배치. 1사이클 = 16장.

```
패턴 A — 대형 좌 + 소형 2×2 (5장, 640px)
┌────────────┬──────┬──────┐
│            │  2   │  3   │  row: 320px
│     1      ├──────┼──────┤
│ (row-span) │  4   │  5   │  row: 320px
└────────────┴──────┴──────┘
grid-template-columns: 2fr 1fr 1fr

패턴 B — 3등분 (3장, 420px)
┌──────┬──────┬──────┐
│  1   │  2   │  3   │  row: 420px
└──────┴──────┴──────┘
grid-template-columns: repeat(3, 1fr)

패턴 C — 소형 2×2 + 대형 우 (5장, 640px)
┌──────┬──────┬────────────┐
│  1   │  2   │            │  row: 320px
├──────┼──────┤     5      │
│  3   │  4   │ (row-span) │  row: 320px
└──────┴──────┴────────────┘

┌─────────────┬─────────────┐
│      1      │      2      │  row: 480px
└─────────────┴─────────────┘

┌────────────────────────────┐
│             1              │  row: 560px
└────────────────────────────┘
grid-template-columns: 1fr
```

### 패턴 사이클 규칙

- **항상 A→B→C→D→E 순서대로** 사이클을 따름 (남은 이미지 수에 따라 패턴을 변경하지 않음)
- 마지막 행에서 이미지가 부족하면 빈 셀을 **컬러 플레이스홀더**로 채움

### 이미지 데이터 구조

```javascript
{src: 'images/gallery/xxx.webp', featured: true}  // featured 태그
{src: 'images/gallery/yyy.webp'}                    // 일반 이미지
```

- `featured: true` 태그가 있는 이미지는 **A/C 패턴의 span(대형 2행) 위치**에 우선 배치
- 게시판 성격이 아닌 편집 가능한 갤러리이므로, 태그로 대형 위치에 적합한 이미지를 지정
- `featured` 풀이 소진되면 일반 이미지로 대체

### 컬러 플레이스홀더 (Extended Palette)

| 이름 | 컬러 | 용도 |
|------|------|------|
| Stone | `#D9D6D2` | 빈 셀 배경 1 |
| Oak | `#E8DFD2` | 빈 셀 배경 2 |
| Espresso | `#D4C8B8` | 빈 셀 배경 3 |
| Olive | `#D5DED0` | 빈 셀 배경 4 |
| Cream | `#F5EFE0` | 빈 셀 배경 5 |

- 플레이스홀더는 5색을 순환 적용
- 호버 효과/클릭 비활성 (`cursor:default`, `pointer-events` 정상이나 클릭 이벤트 무시)

### 라이트박스

- 사진 클릭 → 풀스크린 오버레이 (`rgba(0,0,0,.9)`)
- 좌우 화살표로 이전/다음 이동 (순환)
- 닫기: X 버튼 / 배경 클릭 / ESC 키
- 키보드: ← → 이동, ESC 닫기

### 인터랙션

- 호버: `transform: scale(1.03)` + 반투명 오버레이 (`rgba(0,0,0,.06)`)
- 트랜지션: `transform .5s ease`, `background .3s ease`
