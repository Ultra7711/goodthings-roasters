## 0. 보이스 원칙 (빠른 참조)

| 원칙 | 방향 | 안티패턴 |
|------|------|----------|
| 어조 | 차분하고 자신감 있는, 과시하지 않는 | "최고급!", "한정 수량!", "지금 바로!" |
| 문장 | 짧고 여운이 있는 | 길고 설명적인 문장 |
| 영문 | 소문자 선호, 간결 | 올캡 남용 (BEST COFFEE EVER) |
| 한글 | 존댓말, 부드러운 | 반말, 급한 명령형 |
| 감정 | "가보고 싶다", "괜찮은 곳이다" | "놓치면 후회!", "대박 할인!" |

### 영문 표기 규칙

- 브랜드명: Good Things Roasters (첫 글자만 대문자)
- 슬로건: good things, take time (전부 소문자)
- 섹션 라벨: 올캡 허용 단, 1~2단어에 한정 (MENU, SHOP, BEANS)
- CTA 버튼 영문: Sentence case (Shop now, View all)
- 네비게이션: Title Case (The Story, Good Days)

## 1. 글로벌 요소

### 어나운스먼트 바

```
30,000원 이상 구매 시 무료 배송
```

> 짧고 명확하게. "온라인 상품" 같은 수식어 불필요. 금액과 혜택만.

**시즌/이벤트 대체 예시:**

```
첫 주문 무료 배송
```
```
정기배송 첫 달 10% 할인
```

---

### 글로벌 헤더 네비게이션

| 위치 | 텍스트 |
|------|--------|
| 메뉴 1 | The Story |
| 메뉴 2 | Menu |
| 메뉴 3 | Shop |
| 메뉴 4 | Good Days |
| 아이콘 1 | 검색 (aria-label) |
| 아이콘 2 | 계정 (aria-label) |
| 아이콘 3 | 장바구니 (aria-label) |

> "Work with Us"는 메인 네비에서 제외 — 푸터로 이동. 핵심 4개 메뉴만 노출하여 간결함 유지.

---

### 검색 오버레이

| 요소 | 텍스트 |
|------|--------|
| Placeholder | 무엇을 찾으시나요? |
| 결과 없음 | 검색 결과가 없습니다. |
| 결과 없음 보조 | 다른 검색어로 다시 시도해 보세요. |

---

### 글로벌 푸터

**브랜드 태그라인:**

```
good things, simply roasted.
```

**푸터 컬럼:**

| Column 1: The Story | Column 2: Shop | Column 3: Support | Column 4: Connect |
|---------------------|----------------|--------------------|--------------------|
| Philosophy | Coffee Beans | 자주 묻는 질문 | Instagram |
| Roastery | Drip Bags | 배송 안내 | |
| Location | Cafe Menu | 교환 · 반품 | |
| | | 비즈니스 문의 | |

**하단 법적 정보:**

```
© 2026 Good Things Roasters
이용약관  ·  개인정보처리방침
```

> 사업자 정보(대표명, 사업자번호 등)는 별도 토글 또는 최하단 작은 텍스트로 분리.

## 2. 메인 페이지

### 섹션 1: 히어로 영상

| 요소 | 텍스트 |
|------|--------|
| 메인 카피 | good things, take time |
| 스크롤 유도 | Scroll |

> 히어로에는 슬로건 외 텍스트를 넣지 않는다. 영상이 메시지.
> "Scroll" 라벨은 하단에 작게, 애니메이션으로 존재감만.

---

### 섹션 2: 시즌 배너

**봄 시즌 예시:**

| 요소 | 텍스트 |
|------|--------|
| 시즌 태그 | Spring 2026 |
| 헤드라인 | 봄, 한 잔의 여유. |
| 서브 카피 | 벚꽃이 지기 전에 만나는 시즌 한정 메뉴. |
| CTA | 시즌 메뉴 보기 |

**여름 시즌 예시:**

| 요소 | 텍스트 |
|------|--------|
| 시즌 태그 | Summer 2026 |
| 헤드라인 | 여름, 차갑게 한 잔. |
| 서브 카피 | 시원한 콜드브루와 함께하는 여름 메뉴. |
| CTA | 시즌 메뉴 보기 |

**킬러 메뉴(버터떡) 프로모 예시:**

| 요소 | 텍스트 |
|------|--------|
| 뱃지 | 지금, 매장에서 |
| 헤드라인 | 오늘의 버터떡, 갓 구운 순간. |
| CTA | 메뉴 보기 |

> 시즌 배너는 감성 우선. 짧은 카피 + 이미지가 주역.

---

### 섹션 3: 카페 메뉴 카드

| 요소 | 텍스트 |
|------|--------|
| 섹션 라벨 | MENU |
| 섹션 타이틀 | 오늘, 매장에서. |

**카테고리 카드 3종:**

| 카드 | 타이틀 | 설명 | CTA |
|------|--------|------|-----|
| Coffee | Coffee | 직접 로스팅한 원두로 내리는 한 잔 | 메뉴 보기 |
| Non Coffee | Beverage | 티, 에이드, 그리고 계절 음료 | 메뉴 보기 |
| Dessert | Dessert | 매장에서 직접 굽는 빵과 디저트 | 메뉴 보기 |

> 카드 설명은 1줄. "다양한", "특별한" 같은 수식어 지양. 사실만 담백하게.

---

### 섹션 4: 브랜드 스토리 (Philosophy)

| 요소 | 텍스트 |
|------|--------|
| 섹션 라벨 | OUR STORY |
| 헤드라인 | 좋은 것에는 시간이 필요합니다. |
| 본문 | 빠름보다 바름을 선택합니다. |
| | 정직한 로스팅, 일관된 품질, 그리고 진심. |
| | 그것이 굳띵즈가 지켜온 단 하나의 기준입니다. |
| CTA | The Story |

> 본문은 3줄 이내. 철학을 설명하되 시적으로. "단 하나의 기준"으로 여운.

| 아코디언 | 타이틀 |

| 1 | 상품 정보 |

**배송 안내 내용:**

```
평일 오후 2시 이전 주문 시 당일 발송
기본 배송비 3,000원 · 30,000원 이상 무료 배송
주문 후 1–3 영업일 이내 출고
토·일·공휴일 발송 불가, 다음 영업일 순차 발송
```

**교환 · 반품:**

```
원두 특성상 개봉 후 교환 및 반품 불가
상품 불량·오배송 시 수령 7일 이내 연락
미개봉 단순 변심 반품 가능, 왕복 배송비 고객 부담
```

- **"~하기" 패턴:** 장바구니에 담기, 쇼핑 계속하기, 조회하기 — 부드러운 권유
- **"~하세요" 지양:** CTA에 존댓말 불필요 (본문에서만 사용)
- **급한 톤 금지:** "지금 구매!", "바로 결제!" → "장바구니에 담기", "결제하기"
- **영문 CTA:** 소문자 시작 금지, Sentence case 사용 (Shop now, View all)
